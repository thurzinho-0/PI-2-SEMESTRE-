3.6 Recursos e ferramentas
1. Linguagens de Programação e Frameworks:
•	HTML (HyperText Markup Language): Utilizado para estruturar as páginas web, como cadastro, login, home, carrinho, perfil, entre outras. O HTML é essencial para a criação da estrutura de conteúdo e componentes do site.
•	CSS (Cascading Style Sheets): Usado para o design e estilo do e-commerce, como a criação do layout responsivo, animações e interatividade, e personalização do visual das páginas.
•	JavaScript (JS): Implementado para a interatividade do site, como a validação de formulários, ajuste dinâmico de preços no carrinho, alteração de quantidade de itens e feedback visual para o usuário, além da lógica de adicionar itens ao carrinho e controle de estoque.
•	PHP (Hypertext Preprocessor): Usado no backend para processamento de dados do servidor, como a validação de formulários (login, cadastro), envio de email de recuperação de senha, integração com banco de dados e controle de sessões de usuários.
2. Banco de Dados:
•	MySQL ou MariaDB (presumido): Utilizado para armazenar informações de usuários, produtos, vendas, pedidos, e históricos de compras e vendas. A integração entre PHP e o banco de dados é feita por meio de consultas SQL para recuperação de dados, inserção e manipulação das informações.
3. Frameworks e Bibliotecas:
•	Bootstrap: Para a criação de layouts responsivos e design de componentes reutilizáveis como botões, cards de produto, navegação, formulários. Garantindo a flexibilidade do site em diversos dispositivos.
•	Font Awesome: Usado para ícones, como os de carrinho de compras, usuário, pesquisa e outros elementos interativos do site.
4. Ferramentas de Controle de Versão e Colaboração:
•	Git: Ferramenta de controle de versão usada para gerenciar o código fonte, permitindo que diferentes membros da equipe possam trabalhar simultaneamente no projeto.
•	GitHub: Plataforma usada para colaborar e hospedar o repositório de código, permitindo versionamento de código e integração contínua.
5. Ferramentas de Desenvolvimento:
•	VS Code: Editor de código utilizado para escrever, depurar e editar o código JavaScript, HTML e CSS. Oferece suporte a extensões úteis, como Prettier para formatação do código e Live Server para testar as alterações ao vivo.
•	XAMPP ou WAMP: Usado para configurar o ambiente PHP localmente. Permite o uso de Apache, MySQL e PHP para testar e rodar o backend de maneira eficaz antes de fazer o deploy no servidor.
•	Postman: Utilizado para testar as APIs backend de login, cadastro, recuperação de senha e controle de vendas.
6. Ferramentas de Design e Prototipação:
•	Figma: Para a criação de protótipos de alta fidelidade, wireframes e design do layout. Usado para planejar e ajustar a interface do usuário (UI) e experiência do usuário (UX), assegurando a fluidez do site em todas as plataformas.
